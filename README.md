# حساب المبيعات - نادي الأعوان (نسخة مبسطة)

## الملفات
```
pwa-app/
├── index.html    ← التطبيق كاملاً (HTML + CSS + JS)
├── manifest.json ← إعدادات PWA
├── sw.js         ← Service Worker (يعمل بدون إنترنت)
└── README.md
```

## كيفية الاستخدام
1. ارفع المجلد على أي خادم ويب بسيط (Nginx، Apache، GitHub Pages، Netlify...)
2. افتح `index.html` في المتصفح
3. لتثبيته كتطبيق على الهاتف: افتح في المتصفح ثم "إضافة إلى الشاشة الرئيسية"

## للتشغيل المحلي
```bash
# Python 3
python -m http.server 8080
# ثم افتح: http://localhost:8080
```
